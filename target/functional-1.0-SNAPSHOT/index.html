Usage:

Within a folder (e.g. $PATH/dad-stuff-01/) you will need to have the following files:

  • dad-stuff-01/index.html
  • dad-stuff-01/js/createObjects.js
  • dad-stuff-01/js/testCreateObjects.js

The javascript code of interest is within dad-stuff-01/js/createObjects.js. This javascript file show how to "encapsulate" a small library and how to create several random objects of your choosing.

To see the output, open the index.html in Chrome with the Developer Tools open (Ctl + Shift + I);

Study carefully how the BUtil is setup. testCreateObjects.js shows how to use it.
